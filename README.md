# VR Alchemy – VR Chemistry Education Game

VR Alchemy is a virtual reality educational game designed to help beginners understand chemical reactions through interactive experiments.

The project simulates chemical laboratory experiments in a VR environment, allowing users to explore atoms, observe chemical reactions, and perform simple experiments without requiring real laboratory equipment.

This project was developed as a **team project** at Ritsumeikan University (Information Systems Science and Engineering).

---

## Project Overview

Understanding chemical reactions can be difficult for beginners due to the abstract nature of atoms and molecules.  
VR Alchemy addresses this problem by creating a **visual and interactive learning environment using VR technology**.

Users can interact with atoms and perform simulated chemical experiments to better understand how reactions occur.

The system focuses on visualizing chemical concepts such as:

- atom representation
- flame test reactions
- laboratory equipment
- interactive experimental environment

---

## Technologies Used

- **Unity** – main development platform
- **Blender** – 3D modeling
- **SteamVR** – VR interaction and testing
- **HTC Vive** – VR hardware testing

---

## Features

#### Virtual Chemistry Lab
A VR laboratory environment where users can interact with objects and perform experiments.

#### Atom Visualization
3D models of atoms are used to represent chemical elements in an intuitive way.

Elements implemented in the project include:

- Beryllium
- Fluorine
- Oxygen
- Neon

#### Flame Test Simulation
The game includes a **flame test experiment**, where different metals produce different flame colors:

- Red – Lithium
- Orange-red – Calcium
- Yellow – Sodium
- Green – Barium
- Blue – Copper
- Purple – Potassium

This experiment helps users visually understand how chemical elements can be identified.
